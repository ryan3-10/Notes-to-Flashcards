Notes to Flash Cards

Shuffle

- Input your notes (or any content you want to generate flash cards from) as a file or text below. Make sure the content follows this format:

{Question}?
{Answer}

- The question mark must be included, and the question and answer should be on seperate lines. Blank lines inbetween different questions, or inbetween a question and its answer, are okay.

- Try using the sample text below. You'll notice an alert from the page saying that not all lines in the text have questions and answers. This is because of the top line: "Sample text". The program will still generate the flashcards, ignoring this first line.
